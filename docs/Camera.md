# Camera 相机

## 坐标系

|            | 描述                               | 原点                        |
| :--------- | :--------------------------------- | :-------------------------- |
| 像素坐标系 | 三维空间内的物体在图像平面上的投影 | 左上角（OpenCV 图像坐标系） |
| 图像坐标系 |                                    | CCD 图像平面的中心          |
| 相机坐标系 |                                    | 相机的光心                  |
| 世界坐标系 | 表示空间内物体的绝对坐标           |                             |

## 深度相机

| 类别   | 概述                   |
| :----- | :--------------------- |
| 双目   | 依赖自然光，成本低     |
| 结构光 | 近距离高精度，适合弱光 |
| TOF    | 主动发射光，适合远距离 |

### FOV

Field of View(视场角)，分为 HFOV(水平), VFOV(垂直), DFOV(对角)
